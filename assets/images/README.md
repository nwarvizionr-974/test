# Images

Remplacez les fichiers de démonstration par les images du client en conservant les noms, ou modifiez leurs chemins dans `config/invitation.config.js`.

Recommandations :
- couverture : portrait vertical, idéalement 1600 × 2400 px, WebP ou AVIF ;
- galerie : 1200 à 1800 px sur le grand côté ;
- poids cible : moins de 350 Ko par image ;
- renseigner un texte alternatif fidèle dans la configuration.
