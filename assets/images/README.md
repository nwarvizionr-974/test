# Images

Remplacez les fichiers de démonstration par les photos du client en conservant les noms, ou modifiez leurs chemins dans `config/invitation.config.js`.

- Couverture : portrait vertical, idéalement 1600 × 2400 px, WebP ou AVIF.
- Galerie : 1200 à 1800 px sur le grand côté.
- Poids cible : moins de 350 Ko par image.
