# Modèle d’information RSVP

Les informations transmises par le formulaire sont utilisées uniquement pour l’organisation de l’événement : confirmation de présence, nombre de participants, choix de repas et contraintes alimentaires.

Elles sont accessibles uniquement aux organisateurs de l’événement et supprimées après la durée nécessaire à son organisation.

Contact pour toute demande relative aux données : `contact@votre-domaine.fr`.
